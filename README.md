Tic-Tac-Toe 🎮

A simple Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against a friend in your browser!

Features:

* Two-player mode (X & O)
* Highlights the winning combination
* Reset button to restart the game
* Responsive design
* Tech Stack
* HTML – Game structure
* CSS – Styling & layout
* JavaScript – Game logic & interactivity
  
How to Play:

Click a cell to place your mark. Alternate turns until someone wins or it's a draw. Click Reset to start again.
